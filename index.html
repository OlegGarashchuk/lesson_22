Основи синтаксису JAVASCRIPT

  • Варіант №1) правильно
  • Варіант №2) правильно, але краще після першої фрази поставити крапку з комою.
  • Варіант №3) помилка, потрібно розділити крапкою з комою, і краще другу фразу почати з нової строки.
  • Варіант №4) помилка, у першій фразі незакриті лапки.

Змінні в JAVASCRIPT. Константи. Ключові слова LET CONST VAR

  • Варіант №1) помилка, фрази потрібно розділяти комою, let пишеться один раз, в кінці треба ставити крапку з комою.
  • Варіант №2) написано правильно, але краще написати назву змінної myAge
  • Варіант №3) правильно
  • Варіант №4) правильно
  • Варіант №5) правильно, змінна var видно за межами блоку.

Типи данних JAVASCRIPT. Null String Undefined Boolean Number та інші.

  • Варіант №1) невірно, звернення Фрилансер $(userAge) потрібно брати в зворотні лапки
  • Варіант №2) невірно, поверне значання Infinity
  • Варіант №3) невірно, поверне значання undefined
  • Варіант №4) вірно
